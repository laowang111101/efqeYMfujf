# 农产品销售商城系统 python41

## 项目概述

本项目是一个基于Python的农产品销售商城系统，致力于为用户提供质量可靠、种类丰富的农产品购买平台。系统通过结合前端与后端技术，构建了一个易于管理、便于使用的在线农产品交易环境。

## 技术栈

### 前端技术

- Html
- Css
- Js
- Vue2.0
- Element-ui

### 后端技术

- Python3.7
- Django2.0

## 功能模块

系统包括以下主要功能模块：

- 个人中心
- 用户管理
- 商家管理
- 产品类型管理
- 农产品管理
- 系统管理
- 订单管理

## 系统角色

- 普通用户：浏览商品、下单购买、管理个人账户等。
- 商家：管理商品、处理订单、查看销售数据等。
- 管理员：系统维护、用户管理、商家管理等。

## 运行环境

- 开发工具：PyCharm + VSCODE
- 数据库：MySQL5.7（最低版本要求）
- 数据库管理工具：Navicat10+
- Python版本：Python3.7
- 前端Nodejs版本：14
- 浏览器：谷歌浏览器
